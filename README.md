# 🤖 Telegram RAG Бот для документации Bitrix24 API

## 📖 О проекте

Telegram-бот на базе Yandex GPT с RAG (Retrieval-Augmented Generation) для ответов на вопросы по документации Bitrix24 REST API.

### 🎯 Основные возможности

- **Ответы на вопросы** по документации Bitrix24 REST API
- **Голосовые сообщения** — распознавание через Yandex SpeechKit
- **Гибридный режим** — поиск в документации + ответы как нейросеть
- **История диалогов** — сохранение в SQLite
- **Контекст** — бот помнит предыдущие вопросы
- **Актуальная документация** — 2634 страницы (32 МБ)

---

## 📝 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и начало работы |
| `/help` | Справка по командам |
| `/history` | Показать последние 5 вопросов |
| `/clear` | Очистить историю диалога |

---

## 🎤 Голосовые сообщения

Бот поддерживает голосовые сообщения:
1. Отправьте голосовое сообщение
2. Бот распознает речь
3. Преобразует текст в вопрос
4. Отвечает как на обычный текстовый запрос

---

## 📚 О документации

Бот использует официальную документацию с сайта `https://apidocs.bitrix24.ru/`

- **2634 страницы** спарсено
- **32 МБ** текстовой информации
- Поддерживаются все разделы: CRM, Задачи, Пользователи, Каталог, Чаты, Календарь и др.

##  Переменные окружения (.env)

Для работы бота создайте файл `.env` в корне проекта:

| Переменная | Описание | Пример |
|------------|----------|--------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather | `1234567890:ABC...` |
| `YANDEX_FOLDER_ID` | ID каталога в Yandex Cloud | `b1gq6usts1tm7hihjd9f` |
| `YANDEX_API_KEY` | API-ключ для Yandex GPT | `AQVN...` |
| `YANDEX_SPEECHKIT_API_KEY` | API-ключ для SpeechKit | `AQVN...` |
| `DATABASE_URL` | Строка подключения к БД | `postgresql://user:pass@localhost:5432/db` |
